ray tracer with relativistic rays - aka black hole renderer

requires:
* [glm](https://github.com/g-truc/glm)
* OpenGL version >= 4.5
* C++ version >= 20

build (linux):
$ make [-jN] # if you have N cores

run (slow):
$ make run

change sch_radius to change the amount of bending of light around the mass
